# Image Background Remover - MVP 需求文档

## 1. 项目概述

- **项目名称**: Image Background Remover
- **项目类型**: Web 应用 (图片处理工具)
- **核心功能**: 上传图片，自动移除背景，支持下载透明背景图
- **目标用户**: 需要快速处理产品图、证件照、电商图片的用户

---

## 2. 功能需求

### 2.1 核心功能 (MVP)

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 图片上传 | 支持拖拽或点击上传图片 | P0 |
| 图片预览 | 上传前预览原图 | P0 |
| 背景移除 | 调用 remove.bg API 处理图片 | P0 |
| 结果预览 | 处理后预览透明背景图 | P0 |
| 图片下载 | 下载处理后的图片 (PNG) | P0 |
| 进度提示 | 处理中显示 Loading 状态 | P0 |

### 2.2 支持的图片格式

- 输入: JPG, PNG, WebP
- 输出: PNG (透明背景)

### 2.3 限制 (MVP 阶段)

- 单张图片大小: ≤ 10MB
- 图片尺寸: ≤ 2500x2500 px
- 并发处理: 1 张

---

## 3. 技术方案

### 3.1 架构图

```
用户浏览器
    │
    ▼
┌─────────────────┐
│   前端页面      │  (HTML + JS)
│  (Cloudflare    │  ───────────
│   Pages / 静态) │
└────────┬────────┘
         │ HTTP POST (FormData)
         ▼
┌─────────────────┐
│  Cloudflare     │  (Worker)
│  Worker         │  ───────────
│  (中转服务)     │
└────────┬────────┘
         │ HTTP POST (remove.bg API)
         ▼
┌─────────────────┐
│   remove.bg    │  (第三方 API)
│   API           │
└─────────────────┘
```

### 3.2 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 前端 | HTML + Vanilla JS | 轻量，无框架依赖 |
| 后端 | Cloudflare Worker | 中转 API 请求 |
| API | remove.bg | 图片背景移除 |
| 部署 | Cloudflare Pages + Workers | 免费额度足够 |

### 3.3 API 密钥配置

- **remove.bg API Key**: 通过环境变量配置
- **Cloudflare**: 需要 Workers 和 Pages 服务

## 4. 页面设计

### 4.1 页面结构

```
┌─────────────────────────────────────┐
│           Header                    │
│    "Image Background Remover"      │
├─────────────────────────────────────┤
│                                     │
│         ┌─────────────────┐         │
│         │                 │         │
│         │   上传区域      │         │
│         │  (拖拽或点击)   │         │
│         │                 │         │
│         └─────────────────┘         │
│                                     │
│         [上传图片]  按钮            │
│                                     │
├─────────────────────────────────────┤
│        处理进度: ████████ 80%       │
├─────────────────────────────────────┤
│    原图                    处理后    │
│  ┌─────────┐              ┌─────────┐│
│  │         │              │         ││
│  │   IMG   │     ──▶      │  IMG    ││
│  │         │              │ (透明)  ││
│  │         │              │         ││
│  └─────────┘              └─────────┘│
│                                     │
│         [下载图片]  按钮            │
│                                     │
└─────────────────────────────────────┘
```

### 4.2 交互流程

1. 用户进入页面
2. 拖拽或点击上传图片
3. 前端显示原图预览
4. 点击"移除背景"按钮
5. 显示 Loading 进度
6. Worker 调用 remove.bg API
7. 返回结果并显示预览
8. 用户点击下载 PNG 图片

---

## 5. API 接口设计

### 5.1 remove-bg Worker

**端点**: `/api/remove-bg`

**请求**:
```
POST /api/remove-bg
Content-Type: multipart/form-data

image: <图片文件>
```

**响应**:
```
200 OK
Content-Type: image/png

<PNG 二进制数据>
```

**错误响应**:
```json
{
  "error": "错误描述"
}
```
